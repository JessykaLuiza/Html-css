Criando Sites com HTML e CSS

Aprendendo HTML

HTML Básico

Aqui você aprende a estrutura do HTML e as tags mais usadas:

  • Estrutura mínima de uma página (<!DOCTYPE html>, <html>, <head>, <body>).
  • Títulos e parágrafos (<h1> até <h6>, <p>).
  • Quebras de linha e separadores (<br>, <hr>).
  • Ênfase de texto (<strong>, <em>, <b>, <i>).
  • Listas ordenadas e não ordenadas (<ol>, <ul>, <li>).
  • Links (<a href> com target, rel).
  • Imagens (<img src alt>).
  • Estrutura semântica básica (<header>, <main>, <footer>).

Nesse ponto, o foco é construir páginas simples e estáticas.

HTML Intermediário

Você começa a trabalhar com recursos mais ricos e se preocupar com semântica e acessibilidade:

  • Formulários (<form>, <input>, <textarea>, <select>, <button>, label, fieldset, legend).
  • Tabelas (<table>, <tr>, <td>, <th>, <thead>, <tbody>, <tfoot>).
  • Multimídia (<audio>, <video>, <source>, <track>).
  • Uso correto de tags semânticas (<section>, <article>, <aside>, <nav>).
  • Atributos globais (id, class, title, lang, data-*).
  • Estruturas para acessibilidade (uso de alt, aria-*, ordem lógica).
  • Introdução ao uso de meta tags (SEO e responsividade com viewport).

Aqui você já consegue montar páginas completas, com formulários funcionais e conteúdo organizado.

HTML Avançado

Nesse nível, você entende como o HTML se conecta com CSS, JavaScript e acessibilidade avançada:

  • Estrutura semântica avançada (boas práticas de SEO e indexação).
  • Microdados e Schema.org (dados estruturados para SEO).
  • Formulários avançados: validação nativa com atributos (required, pattern, min, max, step).
  • Elementos interativos modernos (<details>, <summary>, <dialog>).
  • Canvas e SVG para gráficos e animações.
  • Técnicas de acessibilidade avançada (ARIA roles, tabindex, navegação por teclado).
  • Estrutura para PWA (Progressive Web Apps): uso de manifest.json e service workers (embora HTML sozinho não implemente, você prepara a base).
  • Integração com APIs via atributos e elementos (exemplo: <iframe> com sandbox, vídeos embedados).

No avançado, o foco está em escalabilidade, performance, acessibilidade e integração com outras tecnologias.

Resumindo:

  • Básico = montar páginas simples.
  • Intermediário = trabalhar com formulários, tabelas, multimídia e semântica.
  • Avançado = acessibilidade, SEO, microdados, interatividade e integração com outras tecnologias.

Aprendendo CSS

CSS Básico

Aqui você aprende a ligar o CSS ao HTML e os primeiros estilos:

  • Como aplicar CSS: inline, interno (<style>) e externo (.css).
  • Seletores simples: por tag, classe, id.
  • Cores (nomes, hexadecimais, rgb(), hsl()).
  • Unidades absolutas e relativas (px, em, rem, %, vh, vw).
  • Fontes: font-family, font-size, font-weight, line-height.
  • Estilização de texto (color, text-align, text-decoration, letter-spacing).
  • Bordas, margens e preenchimento (border, margin, padding).
  • Backgrounds (background-color, background-image, background-repeat, background-size).
  • Box model (como funcionam as caixas do CSS).

No básico, você já consegue mudar totalmente o visual de uma página estática.

CSS Intermediário

Você começa a dominar layouts, posicionamentos e pseudo-classes:

  • Seletores avançados (filhos >, descendentes, adjacentes +, atributos [attr=value]).
  • Pseudo-classes (:hover, :focus, :nth-child(), :first-child, :last-child).
  • Pseudo-elementos (::before, ::after).
  • Display (block, inline, inline-block, none).
  • Position (static, relative, absolute, fixed, sticky).
  • Float e Clear (conceitos antigos mas importantes).
  • Flexbox (alinhamento e distribuição de elementos).
  • Responsividade: media queries (@media).
  • Cores com transparência (rgba, hsla).
  • Transições (transition).
  • Trabalhar com variáveis CSS (--minha-cor: #333).

No intermediário, você já consegue criar layouts responsivos e interativos.

CSS Avançado

Aqui entram as técnicas mais modernas e poderosas:

  • Grid Layout (linhas, colunas, áreas nomeadas).
  • Animações (@keyframes, animation).
  • Transformações (transform: scale, rotate, translate, skew).
  • Clipping e máscaras (clip-path, mask).
  • Filtros (filter: blur, brightness, contrast).
  • Seletores avançadíssimos (como :not(), :is(), :has()).
  • Variáveis customizadas e temas dinâmicos (dark mode).
  • CSS Functions (calc(), min(), max(), clamp()).
  • Customização de scrollbars.
  • Técnicas de performance (melhorar tempo de renderização).
  • Arquitetura de CSS (BEM, ITCSS, SMACSS).
  • Pré-processadores (SASS/SCSS, LESS) e pós-processadores (PostCSS).
  • CSS moderno para acessibilidade (focus visível, prefers-reduced-motion).

No avançado, você tem controle total sobre design, performance e escalabilidade.

Resumindo:

  • Básico → estilizar textos, cores, espaçamentos, fundos.
  • Intermediário → responsividade, layouts (flexbox), pseudo-classes.
  • Avançado → grid, animações, efeitos visuais, arquitetura de CSS.
