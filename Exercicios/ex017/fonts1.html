Shorthand font

Junta várias propriedades de fonte em uma só linha.

font: [style] [weight] [size] [family];

O que ela pode incluir:

  • font-style → normal, italic, oblique
  • font-variant → normal, small-caps
  • font-weight → 100–900, normal, bold
  • font-size → obrigatório
  • line-height → opcional (vem depois do size, com /)
  • font-family → obrigatório (ex.: Arial, "Open Sans")

Font-Weigth

font-weight é uma propriedade do CSS usada para definir o peso (ou espessura) da fonte de um texto.
Em outras palavras: controla se a letra vai ficar mais fina ou mais grossa.

Formas de usar:

  • Palavras-chave:
    • normal → peso padrão (equivalente a 400)
    • bold → negrito (equivalente a 700)
    • lighter → mais fino que o peso herdado
    • bolder → mais grosso que o peso herdado

  • Números (de 100 a 900):

    • Cada número representa um nível de espessura (nem todas as fontes têm todos os pesos disponíveis):

    • 100 → Thin (bem fino)
    • 200 → Extra Light
    • 300 → Light
    • 400 → Normal (regular)
    • 500 → Medium
    • 600 → Semi-Bold
    • 700 → Bold
    • 800 → Extra-Bold
    • 900 → Black (bem grosso)
