target e rel possui outros atributos dentro além de blank e external.

  • Tag target
    • _blank→ Abre o link em uma nova aba ou janela
    • _self→ Abre o link na mesma aba (padrão, mesmo sem escrever)
    • _top→ Abre o link no topo da janela, ignorando qualquer frame
    • _parent→ Abre o link na janela pai, usada quando se está dentro de frames
    • nome-do-frame→ Abre o link em um frame específico (caso esteja usando frames)
  • Tag rel
    • next→ Link para a próxima parte do conteúdo ou documento
    • prev→ Link para a parte anterior do conteúdo
    • author→ Link para a página ou site do autor do artigo atual
    • external→ Indica que o link leva para fora do seu site
    • nofollow → Informa aos buscadores não seguir esse link
    • noopener → Evita que a nova aba/janela tenha controle sobre a página original
    • noreferrer → Impede que o site de destino receba dados da origem (referência)
