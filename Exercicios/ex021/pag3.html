Explicando o border-image

O border-image é uma propriedade do CSS que permite usar uma imagem como a borda de um elemento em vez de uma cor ou estilo sólido. É mais avançado que o border comum, porque você consegue fatiar uma imagem e aplicar essas partes nas bordas do elemento (cantos e laterais).

A propriedade border-image é composta por várias subpropriedades que controlam diferentes aspectos da borda de imagem:

  • border-image-source: Especifica a imagem que será usada como borda.
  • border-image-slice: Define como a imagem será dividida em nove partes (quatro cantos, quatro lados e o centro). Você pode especificar valores em pixels ou porcentagens para controlar onde a imagem é cortada.
  • border-image-width: Controla a largura da borda de imagem. Pode ser definido em pixels, porcentagens ou usando palavras-chave como "auto".
  • border-image-outset: Define o quanto a borda de imagem deve se estender além da borda do elemento.
  • border-image-repeat: Especifica como as partes da imagem serão repetidas ou esticadas para preencher as bordas. As opções incluem "stretch" (esticado), "repeat" (repetido) e "round" (ajustado para caber).

Exemplo de uso:

        
            border: 10px solid transparent; /* Define a largura da borda */
            border-image-source: url('borda.png'); /* Imagem da borda */
            border-image-slice: 30; /* Fatiamento da imagem */
            border-image-repeat: repeat; /* Repetição da imagem */

No mode shorthand:

            border: 10px solid transparent;
            border-image: url('borda.png') 30 repeat;

Bancos de imagens / vetores gratuitos

  • Pixabay → Pesquise por "frame border png" ou "ornamental border".
  • Freepik → Muitos vetores decorativos (alguns exigem atribuição).
  • Vecteezy → Tem bastante coisa em PNG e vetor.
  • SVG Repo → Padrões e bordas em SVG (ótimos para não perder qualidade).

Ferramentas de padrão online

  • CSS Gradient → Gera gradientes que você pode salvar e usar como borda.
  • Pattern Monster → Gera padrões SVG que podem virar border-image.
  • Transparent Textures → Padrões leves para aplicar como borda.
